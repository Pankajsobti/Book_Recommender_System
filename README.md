# 📚 Book Recommendation System

This project is a **Machine Learning based Book Recommendation System** that suggests books to users based on their preferences and past ratings.

---

## 🎯 Objective

To build an intelligent system that recommends books based on user preferences using machine learning techniques.

---

## 🚀 Features

- Personalized book recommendations  
- Collaborative filtering approach  
- Data preprocessing and cleaning  
- Similarity-based suggestions  

---

## 🛠️ Tech Stack

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Jupyter Notebook  

---

## 📂 Dataset

The project uses the following datasets:
- Books dataset  
- Users dataset  
- Ratings dataset  

> Due to GitHub file size limitations, the dataset is not fully included.

👉 Download dataset from Kaggle:  
https://www.kaggle.com/datasets/arashnic/book-recommendation-dataset

---

## 📊 How it Works

1. Load datasets (Books, Users, Ratings)  
2. Clean and preprocess data  
3. Create user-item matrix  
4. Apply similarity algorithm  
5. Recommend books based on user behavior  

---

## ▶️ How to Run

1. Download the dataset from the link above  
2. Place all CSV files in the project folder  
3. Open the notebook: Book_recommender_system.ipynb
4. Run all cells step by step  
5. Enter a book name to get recommendations
