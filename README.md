# 📚 Book Recommendation System

This project is a **Machine Learning based Book Recommendation System** that suggests books to users based on their preferences and past ratings.

---

## 🚀 Features

- Personalized book recommendations
- Collaborative filtering approach
- Data preprocessing and cleaning
- Similarity-based suggestions

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

---

## 📂 Dataset

The project uses the following datasets:
- Books dataset
- Users dataset
- Ratings dataset

> Note: Dataset may not be included due to large file size.

---

## 📊 How it Works

1. Load datasets (Books, Users, Ratings)
2. Clean and preprocess data
3. Create user-item matrix
4. Apply similarity algorithm
5. Recommend books based on user behavior
